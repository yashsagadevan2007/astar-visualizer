How A* Works

A* combines the real cost to reach a node with a smart estimate of how far the goal is — making it faster and smarter than any blind search.

f(n)
Total Score
Priority of a cell. A* always picks the cell with the lowest f(n) next.
=
g(n)
Real Cost
Exact steps taken from the start to reach this cell.
+
h(n)
Heuristic
Estimated distance to the goal using Manhattan distance: |Δrow| + |Δcol|.
01

Start node

Add the start cell to the Open List with f = 0 + h(start).

02

Pick the best

Always pick the cell with the lowest f(n) from the Open List.

03

Explore neighbors

Calculate g, h, and f for each neighbor. Update if a better path is found.

04

Repeat until goal

Move processed cells to Closed List. Stop when the goal is reached.
